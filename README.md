# AjLisp

Lisp interpreter written in C#.

## Origin

It's a copy and adaptation of https://code.google.com/p/ajlisp/ (the [AjLisp solution](https://code.google.com/p/ajlisp/source/browse/#svn%2Ftrunk%2FAjLisp))

## Contribution

Feel free to [file issues](https://github.com/ajlopez/AjLisp) and submit
[pull requests](https://github.com/ajlopez/AjLisp/pulls) — contributions are
welcome.

If you submit a pull request, please be sure to add or update corresponding
test cases, and ensure that tests continue to pass.
